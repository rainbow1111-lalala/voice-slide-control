Setup console

1 Choose recognition

2 Add API key

① Open the provider console and create a key → ② paste below → ③ click Test connection.

Advanced: custom base_url / model (point at a self-hosted whisper server)

2 Local model

The small local model (~42MB) is fully offline and free, but less accurate on fast short commands. For reliability, use the cloud API.
